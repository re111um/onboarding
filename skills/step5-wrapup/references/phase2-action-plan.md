# Phase 2: 팀 킥오프 준비 조언

## EXPLAIN

### 팀 합류 전 팁

온보딩 캠프가 끝나면 팀에 합류합니다. 팀 킥오프는 **팀 리드가 직접 진행**하는 영역이에요.
여기서는 킥오프 전에 알아두면 좋은 **가벼운 조언**만 드릴게요.

**킥오프 미팅 전에 준비하면 좋은 것:**
- 온보딩에서 배운 내용 중 **팀 리드에게 물어보고 싶은 것** 1~2개 정리
- 팀 Slack 채널의 최근 대화를 한번 훑어보기 (분위기 파악)
- 첫 주에 해보고 싶은 것이 있다면 메모해두기

**킥오프에서 팀 리드가 안내해줄 것들:**
- 팀 업무 프로토콜 (소통 방식, 미팅 루틴 등)
- 현재 진행 중인 프로젝트와 내 역할
- 첫 주 업무 배정

> 구체적인 업무 세팅과 액션플랜은 팀 리드와 킥오프에서 함께 잡아보세요!

## EXECUTE

킥오프 준비는 간단해요:

1. 온보딩에서 궁금했던 것 중 **팀 리드에게 물어볼 질문** 1~2개를 정리해보세요
2. 팀 Slack 채널을 한번 훑어보세요 (Claude에게 요약 요청 가능)

> 정리했으면 "완료" 또는 "다음"이라고 입력해주세요!

## CHECK

**확인**: "킥오프에서 팀 리드에게 물어보고 싶은 것이 있나요?"
- 오픈형 — 있으면 좋고, 없어도 괜찮다. "아직 모르겠어요"도 OK
- 격려: "킥오프에서 팀 리드가 잘 안내해줄 거예요. 궁금한 건 그때 편하게 물어보세요!"
