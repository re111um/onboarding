# Step 4: Build Your First Skill — 폴더 컨텍스트

## 비개발자 지원 핵심

이 Step의 대상은 개발자가 아닌 신규입사자가 대부분이다.
Git 명령어를 직접 입력하는 것이 아니라 Claude Code에게 시키는 것임을 강조한다.

## GitHub 설정 흐름

1. `gh auth login` → GitHub 계정 인증
2. `{{GITHUB_ORG}}` org 초대 확인 (없으면 HR에게 요청)
3. Notion의 GitHub 사용 가이드(`github_guide` ID) 사전 읽기
4. 브랜치 → 커밋 → PR 모두 Claude Code가 대행

Phase 2의 EXECUTE에서 팀별 아이디어 리스트를 참조한다.
