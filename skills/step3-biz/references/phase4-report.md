# Phase 4: Biz 리포트 작성

## EXPLAIN

### 리포트를 왜 작성하나요?

Phase 1에서 BM/경쟁력/시장을, Phase 2에서 시나리오 퀘스트를, Phase 3에서 팀별 심화를 배웠습니다.
이제 이 내용을 간단히 정리하여 **내 직무가 회사에 어떻게 기여하는지** 한 장으로 만듭니다.

이 리포트는:
- 본인의 비즈니스 이해도를 정리하고
- 직무 목표 설정의 기초 자료가 되며
- HR Lead({{HR_LEAD}})와 직속 리더에게 공유됩니다

### 리포트 구조

1. **BM 한 줄 요약** -- {{SERVICE_NAME}}가 돈 버는 방법을 한 문장으로
2. **경쟁 포지셔닝** -- 경쟁사 대비 {{SERVICE_NAME}}의 차별점
3. **시나리오 퀘스트 회고** -- 가장 인상적이었던 시나리오와 배운 점
4. **내 직무 x BM 연결** -- 내 업무가 고객 여정의 어느 단계에 기여하는지
5. **입사 후 3개월 목표** -- BM 이해를 바탕으로 한 직무 목표 1~3개
