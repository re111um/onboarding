# Phase 3: Slack 인사 보내기

## EXPLAIN

### Slack 인사 보내기

오늘 아침에 TV 앞에서 전체 구성원들에게 소개 시간을 가지셨지만, 휴가나 외근, 회의 중이라 자리에 없었던 분들이 계실 수 있어요.
Slack에 간단히 인사를 남겨볼까요?

> **에이전트 지시**: 이 실습의 실제 목적은 **Claude Code로 Slack 메시지를 보내는 경험**을 해보는 것이다.
> 따라서 자기소개를 길게 작성할 필요 없이, 간단한 인사 한마디면 충분하다.
> "함께 정교하게 다듬기" 같은 과도한 과정은 불필요 — 짧고 가볍게 진행한다.

---

## EXECUTE

Claude Code로 `#{{CHANNEL_ANNOUNCEMENTS}}` 채널에 간단한 인사를 보내봅니다.

예시:
```
#{{CHANNEL_ANNOUNCEMENTS}} 채널에 인사 메시지 보내줘.
"안녕하세요! 오늘부터 [팀명]에서 [직무]를 맡게 된 [닉네임]입니다. 잘 부탁드립니다!"
```

> 이렇게 Claude Code를 통해 Slack 메시지를 보낼 수 있어요!
> 앞으로 온보딩 과정에서도 Slack 연동을 자주 활용하게 됩니다.
