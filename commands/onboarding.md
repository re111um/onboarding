# /onboarding — 피처링 신규입사자 온보딩

## 목적

피처링 신규입사자가 회사에 완전히 적응하고,
AI Native로 성장할 수 있도록 돕는 7단계 인터랙티브 온보딩.

## 진입 시 처음 할 일

1. 환영 인사를 전한다
2. 본인의 닉네임, 소속 본부/팀, 직무, 입사일을 물어본다
3. 전체 온보딩 로드맵을 안내한다
4. Step 0부터 시작한다

## 온보딩 흐름

```
Step 0 — 환경 설정
    └ @.claude/skills/step0-setup/SKILL.md

Step 1 — General (컬쳐, 도구, Slack 인사)
    └ @.claude/skills/step1-general/SKILL.md

Step 2 — Product (피처링 플랫폼 이해)
    └ @.claude/skills/step2-product/SKILL.md
    └ 완료 시 → Poza에게 커피챗 DM

Step 3 — Biz (BM + 본부별 시나리오)
    └ @.claude/skills/step3-biz/SKILL.md
    └ 완료 시 → Eve에게 커피챗 DM

Step 4 — Build Skill (보류)
    └ @.claude/skills/step4-skill/SKILL.md
    └ 현재 구조만 존재. 실제 도입 고도화 후 활성화.

Step 5 — Wrap Up (회고 + 킥오프 조언)
    └ @.claude/skills/step5-wrapup/SKILL.md
    └ VoC → Thomas에게 Slack DM

Step 6 — 플랫폼 체험 (보류)
    └ @.claude/skills/step6-platform/SKILL.md
    └ 개발팀(Earl) 협업 후 콘텐츠 확정 예정
```

## 완료 시 자동 액션

- Step 5 완료 → 팀 리드 + Thomas에게 온보딩 완료 Slack DM

## STOP PROTOCOL 적용
