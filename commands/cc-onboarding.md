# /cc-onboarding — 피처링 기존직원 Claude Code 온보딩

## 목적

피처링 전사 AI Native 전환을 위한 기존 직원 대상 Claude Code 온보딩.
현재 Claude Code는 개발팀 중심으로 사용 중이며, 이 온보딩은 전사 확대를 목표로 한다.

## 진입 시 처음 할 일

1. 본인의 닉네임과 소속 본부/팀을 물어본다
2. 현재 Claude 사용 경험 여부를 확인한다 (미사용 / claude.ai만 / Claude Code 경험 있음)
3. 아래 트랙 구조를 안내하고 시작한다

## 트랙 구조

```
Phase 1 (필수) — 설치 + 기초 활용
    └ @.claude/skills/cc-phase1-install/SKILL.md

Phase 2 (필수) — 본부별 실무 활용 실습
    └ @.claude/skills/cc-phase2-practice/SKILL.md

Phase 3 (선택) — 스킬 제작
    └ @.claude/skills/cc-phase3-skill/SKILL.md
    └ "여기까지만 할게요"라고 하면 언제든 종료 가능
```

## 완료 시 자동 액션

- Phase 2 완료 → Thomas(HR)에게 완료 Slack DM 발송
- Phase 3 완료 → Thomas(HR)에게 스킬 완성 알림 DM 발송
- 전체 완료 후 VoC 수집 → Thomas에게 전달

## STOP PROTOCOL 적용

모든 Phase는 CLAUDE.md의 STOP PROTOCOL을 따른다.
